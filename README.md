Blogging Platform

A clean, aesthetic, and functional blogging platform where users can create, edit, view, and delete blog posts. This project focuses on backend fundamentals, routing, form handling, and dynamic rendering using EJS.

Features

Create new blog posts

View all published posts

Edit existing posts

Delete posts

Clean card-based UI layout

Responsive grid (3 posts per row)

Server-side rendering using EJS

Organized folder structure

Installation & Setup
1. Clone the repository
git clone https://github.com/your-username/blogging-platform.git
cd blogging-platform
2. Install dependencies
npm install
3. Run the server
node index.js

or

npm start
4. Open in browser
http://localhost:3000
